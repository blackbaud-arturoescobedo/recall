# OBS Overlay Setup (Recall)

## File
- `c:/recall/recall-overlay.html`

## How to use in OBS
1. In OBS, add source: **Browser**.
2. Enable **Local file**.
3. Browse to `c:/recall/recall-overlay.html`.
4. Set Width: `1920`, Height: `1080`.
5. Keep the source above your screen capture source.

## Suggested usage
- Show overlay for the first 3-5 seconds (opening).
- Hide during main demo.
- Show again for closing.
